### **9. Q&A Preparation**

**Common Questions & Answers:**

**Q: Is this replacing the instructor?**
A: No, it's a tool to support students. Instructors still set goals, provide feedback, and guide the learning process. The AI assists with structure and objective errors, not content evaluation.

**Q: Can students cheat with AI?**
A: The tool is designed to teach writing skills, not replace thinking. It helps with structure and mechanics, but students must still develop their own ideas and arguments.

**Q: What about data privacy?**
A: All data is stored securely in the Moodle database. The AI API interactions can be configured according to institutional privacy policies.

**Q: Can it be customized?**
A: Yes, instructors can set goals, select templates, and provide instructions. The system is flexible and adaptable.

**Q: What writing types does it support?**
A: Currently supports Argumentative Essays, Comparative Essays, and Lab Reports. More templates can be added.
